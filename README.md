## PHP_BLOG

PHP를 사용해 웹 페이지를 제작하였습니다. 댓글 기능, 게시판, 로그인, 회원가입 기능을 넣었습니다.<br>
MyAdmin 에 접속한 후 MySQL을 이용해 데이터베이스에 접속하여 데이터들을 수정, 삭제, 복사 등을 할 수 있습니다.
